# Espaço Clínico Florescer – Site + Ficha de Anamnese (Formspree)

Arquivos incluídos:
- `index.html` → Landing page da clínica.
- `anamnese.html` → Ficha de anamnese infantil COMPLETA integrada ao Formspree.
- `obrigado.html` → Página de agradecimento.
- `README.md` → Instruções.

## Endpoint do Formspree

O formulário já está apontando para o endpoint:

> https://formspree.io/f/xblnkggy

e redireciona para:

> https://espaco-clinico-florescer.vercel.app/obrigado.html

Depois de publicar o site na Vercel com o nome `espaco-clinico-florescer`, o fluxo estará completo.

Passos rápidos:

1. Criar repositório no GitHub e enviar estes arquivos.
2. Na Vercel, importar o repositório e fazer o deploy.
3. Testar o preenchimento da ficha acessando:
   https://espaco-clinico-florescer.vercel.app/anamnese.html

Sempre que a ficha for enviada, você receberá um e-mail via Formspree.
